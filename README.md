📋 Task Manager App

A modern and user-friendly task management web application built with Next.js, React, and Tailwind CSS. This app allows users to create, manage, and track tasks efficiently with a clean and responsive UI.

🚀 Live Demo

🔗 https://from-gilt.vercel.app/tasks

🛠️ Tech Stack
⚡ Next.js (App Router)
⚛️ React.js
🎨 Tailwind CSS
🧠 Server Actions (Next.js)
📦 Local Storage / API (JSON / Custom)
